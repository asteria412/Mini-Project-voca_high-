## 📊 Deep Research: 2025 HSK 5급 출제 경향 분석 및 기술적 반영

본 프로젝트는 단순한 단어 암기 앱을 넘어, **2024-2025년 최신 HSK 5급 출제 트렌드 심층 분석**을 기반으로 설계되었습니다. 
'감(Feeling)'이 아닌 '철저한 논리(Logic)'로 합격하는 전략을 LLM 프롬프트 엔지니어링에 녹여냈습니다.

### 1. 작문 1부분 (어순 배열) : "감(感)에서 구조(Structure)로"
> **[분석 Insight]** 최신 경향은 한국어 해석에 의존하는 '의미 중심 접근'을 저격합니다. 
문법적 격식과 **호응 관계(Collocation)**를 모르면 풀 수 없는 함정 문제가 다수 출제됩니다.

* **🛠️ Tech Implementation (AI 출제 로직)**
    * **호응 관계 분리:** AI가 문장을 생성할 때, `提高(향상시키다)` - `水平(수준)`과 같은 짝꿍 단어를 의도적으로 멀리 떨어뜨려 배치하여 구조적 파악 능력을 테스트합니다.
    * **복잡한 관형어:** 주어/목적어를 수식하는 긴 관형어(Phrase + 的) 구조를 생성하여, '的'의 위치를 찾는 심화 훈련을 제공합니다.
    * **논리 접속사:** `因此(따라서)`, `但是(그러나)` 등 문맥의 흐름을 결정하는 접속사를 포함시켜 문장 논리력을 평가합니다.

### 2. 작문 2부분 (99번 제시어 작문) : "논리적 연결과 호응"
> **[분석 Insight]** 서로 상관없어 보이는 5개의 단어를 하나의 스토리로 엮어야 합니다. 단순 나열은 감점 대상이며, **논리적 연결(접속사)**과 **단어 간의 호응**이 핵심 평가 요소입니다.

* **🛠️ Tech Implementation (AI 채점 로직)**
    * **스파르타 모드:** 가이드 없이 단어장 기반의 랜덤 5단어를 던져주어 실전과 동일한 환경을 제공합니다.
    * **심층 평가:** 단순히 단어 포함 여부만 보는 것이 아니라, LLM이 **"억지스러운 연결인지", "올바른 호응구(Collocation)를 썼는지"**를 중점적으로 피드백합니다.

### 3. 작문 2부분 (100번 그림 작문) : "4대 빈출 테마와 만능 서사"
> **[분석 Insight]** 그림 문제는 **비즈니스, 일상, 스포츠, 학습** 등 4대 빈출 카테고리 내에서 반복 출제됩니다. 
상황 묘사에 그치지 않고 자신의 감정이나 경험(**만능 서사**)으로 마무리해야 고득점이 가능합니다.

* **🛠️ Tech Implementation (AI 시나리오 생성)**
    * **전략적 출제:** AI가 무작위로 문제를 내지 않고, 리서치된 **4대 핵심 테마** 내에서 `격식체(비즈니스)`, `심리묘사(스포츠)` 등 테마별 필수 공략 포인트를 반영하여 상황을 묘사합니다.
    * **이미지 생성(DALL-E 3):** 텍스트 묘사를 바탕으로 실전과 유사한 삽화를 실시간으로 생성하여 시각적 훈련을 돕습니다.
    * **전략 피드백:** 채점 시 **"만능 서사(자신의 생각/감정)가 포함되었는가?"**를 체크하여 단순 묘사를 넘어선 고득점 답안을 유도합니다.
